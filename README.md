## Get Started:
1. **Clone the Repository:**
```bash
git clone https://github.com/MrSentinel137/tot-fiesta.git
cd tot-fiesta
```

2. **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
     OR
    
    ```bash
    pip3 install -r requirements.txt
    ```

3. **Run the Code:**
 ```bash
 streamlit run ToT.py
```
